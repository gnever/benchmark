功能描述
======

针对[goim](https://github.com/Terry-Mao/goim)的简单压测工具


参数描述
======
*  -debug int
    	debug
*  -h string
    	url (default "127.0.0.1:8090")
*  -ht int
    	心跳间隔 (default 2)
*  -m int
    	Number of clients (default 1)
*  -path string
    	path (default "sub")
*  -roomid int
    	roomid (default 123)
*  -s int
    	start uid (default 1)
*  -st int
    	向服务端push消息间隔 单位 s, 0 为不推送
*  -stst int
    	数据统计区间 (default 5)
